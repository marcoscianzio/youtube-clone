# Youtube Clone

## How to run the project
1. start mysql service
2. create a mysql database called `youtube`
3. start `redis-server`

### `server`
```
cd server
yarn
yarn watch
yarn dev
```
### `web`
```
cd web
yarn
yarn dev
```
